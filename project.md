
# Current Project

## Swoop Adventure

Swoop Adventure is a platformer game developed using Unity and C#. The goal of this project is to create a playable prototype that demonstrates core 
gameplay mechanics such as player movement, level interaction, and obstacle navigation.

### Features

- Player movement system
- Platforming mechanics
- Collision detection
- Basic level design

### Development Tools

- Unity Game Engine
- C#
- Visual Studio Code
- GitHub for version control

### What I Learned

During this project I learned how to implement movement mechanics, work with Unity’s physics system, and structure a project using version 
control through GitHub. The project helped me better understand
the process of prototyping gameplay systems and testing mechanics in a playable environment.
