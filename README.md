#  Swoop Adventure

##  Game Overview
Swoop Adventure is a 2D platformer where players control a bird named Swoop. The goal is to navigate platforms, collect points, and avoid hazards while surviving as long as possible.

---

##  Controls

- A / Left Arrow → Move Left  
- D / Right Arrow → Move Right  
- Space → Jump  

---

##  Game Mechanics

- Player movement using Rigidbody2D physics  
- Jump system with grounded detection  
- Score system that increases over time or via collectibles  
- Lives system with Game Over screen  
- Animations for idle, running, and jumping  
- UI elements including score and lives display  

---

## Special Features

- Smooth animation transitions using Animator  
- Game Over screen with fade-in effect  
- High score system (DatabaseManager)  
- Sound effects for jumping  
- Player sprite flipping based on direction
