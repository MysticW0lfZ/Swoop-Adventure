Swoop Adventure

Project Overview



Swoop Adventure is a 2D platformer developed in Unity 6. The player navigates through a forest environment, avoids hazards, collects items, and manages health while trying to complete the level.



Current Features



Player movement system



Collectible and score tracking



Health system with damage



Kill zone and respawn system



Game Over UI with restart functionality



Level completion trigger



Milestones



Milestone 1

Core movement and level setup



Milestone 2

Gameplay loop implemented (collect, survive, finish level)
